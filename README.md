## Installing
To install the pack you will need [`packwiz-installer`](https://github.com/packwiz/packwiz-installer-bootstrap/releases), for more information read: https://packwiz.infra.link/tutorials/installing/packwiz-installer/#creating-a-multimc-instance-for-your-modpack

## Development
1. Ensure that you have [pakcwiz](https://github.com/packwiz/packwiz) installed and in your path
2. Run `scripts/setup-hooks.sh` to install git hooks ensuring that `pack.toml` and `index.toml` are propertly updated
